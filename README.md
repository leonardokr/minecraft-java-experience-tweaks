# Metalion's Experience Tweaks

A NeoForge mod for Minecraft Java 26.1.2 with a suite of configuration tweaks for experience, enchanting, anvil, fishing, and enchantments.

## Features

- Keep XP on death (per-player configuration).
- Direct mob XP insertion into the player.
- Item-based enchanting and custom per-player enchantment cooldown system.
- Anvil modifications (bypass "Too Expensive" limit, use item costs instead of levels, enchantment extraction, configurable anvil durability).
- Automatic fishing and automatic recasting client-side helpers.
- Allow combining normally incompatible vanilla enchantments (weapons, armor, crossbows, and tridents).
- Client-side preference to use Riptide anywhere regardless of weather or water.
- OP button to unlock all recipes for all players.

## Configuration

Settings can be customized via standard NeoForge config files:
- Client-side preferences: config/experiencetweaks-client.toml
- Server-side mechanics: config/experiencetweaks-server.toml

An in-game GUI screen is also available for OPs and players to configure client and server settings directly.

## Installation

1. Install NeoForge.
2. Place the mod jar file in the mods folder.
