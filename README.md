# Metalion's Experience Tweaks

A custom NeoForge mod developed for our personal Minecraft server, featuring various quality-of-life tweaks and mechanical adjustments designed to tailor the gameplay to our liking.

## Configuration

Settings can be customized via standard NeoForge config files:
- Client-side preferences: `config/experiencetweaks-client.toml`
- Server-side mechanics: `config/experiencetweaks-server.toml`

An in-game GUI screen is also available (defaulting to the shared `K` hotkey) for OPs and players to configure client and server settings directly.

## Installation

1. Install NeoForge.
2. Place the mod jar file in the mods folder.
